vector-along {rlang}R Documentation

Create vectors matching the length of a given vector

Description

These functions take the idea of seq_along() and generalise it to creating lists (list_along) and repeating values (rep_along). Except for list_along() and raw_along(), the empty vectors are filled with typed missing values.

Usage

lgl_along(.x)

int_along(.x)

dbl_along(.x)

chr_along(.x)

cpl_along(.x)

raw_along(.x)

bytes_along(.x)

list_along(.x)

rep_along(.x, .y)

Arguments

.x

A vector.

.y

Values to repeat.

See Also

vector-len

Examples

x <- 0:5
rep_along(x, 1:2)
rep_along(x, 1)
list_along(x)
