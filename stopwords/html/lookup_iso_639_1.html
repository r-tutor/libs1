lookup_iso_639_1 {stopwords}R Documentation

return ISO-639-1 code for a given language name

Description

Looks up the two-character ISO-639-1 language code for a given language name.

Usage

lookup_iso_639_1(language_name, source)

Arguments

language_name

character; name of a language

source

the short name for a language source, e.g. "snowball"
